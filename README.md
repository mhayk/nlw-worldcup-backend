# NLW WORLDCUP BACKEND

## Prisma
```bash
$ npx prisma init --datasource-provider SQLite
$ npx prisma migrate dev
```

### Prisma Studio
```bash
$ npx prisma studio
```

## Prisma Seed
```bash
$ npx prisma db seed
```

### Prisma ERD Generator
```bash
$ npm i prisma-erd-generator @mermaid-js/mermaid-cli -D
$ npx prisma generate
```
